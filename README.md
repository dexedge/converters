# Austrian converters

A Javascript-based web converter for historical Austrian units. Currently includes:

* length
  * österreichische Postmeile, kilometers, miles
  * Viennese Klafter, Fuß, Zoll
  * Salzburg Klafter, Fuß, Zoll
  * Meters, centimeters
  * Feet, inches

* temperature
  * Réaumur
  * Celsius
  * Fahrenheit
